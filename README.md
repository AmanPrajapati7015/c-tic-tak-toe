# Tic-Tac-Toe (C++)

A terminal-based Tic-Tac-Toe game in C++ with an unbeatable AI using the Minimax algorithm. Uses CRTP to avoid virtual function overhead and improve performance.

---

## Features
- Minimax AI (optimal play)
- CRTP (compile-time polymorphism)
- Simple terminal interface
- 0-based indexing input `(i, j)`

---

## Setup & Run

### Clone the repo
```bash
git clone https://github.com/AmanPrajapati7015/c-tic-tak-toe/

cd c-tic-tak-toe
```
### Compile and run
```bash
g++ -std=c++17 main.cpp board.cpp

./a.out
```
